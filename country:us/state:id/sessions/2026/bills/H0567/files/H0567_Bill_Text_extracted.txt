34 SECTION 1. IDAHO CODE CLEANUP -- LEGISLATIVE INTENT. It is the intent
35 of the Legislature to ensure that the state laws provided in Idaho Code are
36 streamlined, up-to-date, and essential for the citizens of Idaho, while best
37 serving the public health, safety, and welfare. In accordance with the Idaho
38 Code Cleanup Act, following comprehensive review, the Legislature has iden-
